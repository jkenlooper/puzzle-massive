Buttons

Button styles modified from suitcss-components-button 6.0.2. Variants inspired by Material Design.

Button style can be applied to inline and block level elements. Commonly set on anchor, button, and input elements. The default style of the Button class is an outlined button with transparent background and no shadow.

  • span (should always have is-disabled class)
  • div
    (should always have is-disabled class)
  • Anchor

The u-elevation[0-5] utility class is not recommended to use on outlined buttons. Only use them on contained buttons when needing to bump up the elevation.

Button UI States

Disabled

Disabled buttons are set with the .is-disabled class or the disabled attribute. Note that the span element is used for disabled buttons that would normally be a link.

and is-disabled Class

Button Modifiers

Follows SUITCSS guidelines and use both the Button and modifier class on the element.

  • Button--text
  • Button--contrast
  • Button--primary
  • Button--secondary
  • Button--tertiary
  • Button--accent
