Help

Problems with the site? Need help with a puzzle? Got a cool idea? Why are you reading this page when you could be puzzling?

  • Online chat with other players at Discord
  • Send a bug report to puzzle-bug@massive.xyz
  • Or create an issue on the project page for Puzzle Massive

I made up some questions and answers if the answers that you have questions for are found here… then maybe that is useful.

Bug reports

Please submit any issues with the site at this bug submission form or email puzzle-bug@massive.xyz if you don't want to add it directly to the issue board on the project page. Sensitive security issues and such should be emailed or use the bug submission form.

I know there are a lot of things that can be improved and your detailed report will help. You may also submit various feature requests for ideas you have that would make the puzzle site more fun.
